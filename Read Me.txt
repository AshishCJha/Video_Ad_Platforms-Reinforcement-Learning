Objective - To check which Ad Platforms best amoung others. Similar to A/B testing of marketing before starting any big campaign 

Machine Learning - Will be using Reinforcement Learning

Algorithm Used - Upper Confidence Bound, Thompson Sampling

Data Description - Data consist of 1 Video Ad which will be shown on 4 popular Ad Platforms to 149 people. CTR above 1% has been considered as 1 and below 1% as 0

Repository - This folder contain the raw data, codes and screenshots
